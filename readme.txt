== Description ==
Influx FB Like Box like box display facebook like box. it enable Facebook Page owners to attract and gain Likes from their own website. The Like Box enables users to see page likes, recent posts from the page and Like button and which of friends like this page too.[youtube https://www.youtube.com/watch?v=xf3tHig7DAM]

= Options =
1. Facebook Page URL - enter an URL address of your Facebook fan page.2. Width - set the width of the like box in pixels.3. Height - Set the height of like box in pixels.4. Show Faces - choose show or hide faces.5. Posts - choose show or hide lastest posts from your facebook fan page.6. Cover Photo - choose show or hide Cover Photo.

== Installation ==

= Option 1 =
1. Upload the entire Influx FB Like Box Like Box folder to your wp-content/plugins/ directory.2. Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2 =
1. Download Influx FB Like Box Like Box.2. Go to wordpress dashboard > plugin > add new and click on upload.3. Browse Influx FB Like Box Like Box and click on upload button.4. Then click on Activate plugin.
